1회성 알람이 울린 후(또는 끈 후)에는 목록에서 사라져야 한다.
토스트 메시지: 알람 저장/삭제/끄기 등 주요 동작마다 토스트 표시.
시간 선택 UI: 오전/오후, 시, 분 3개 피커로 구성. 현재 시간이 기본값.
화면 전환 시 흰색 깜빡임: 전체적으로 수정 필요 — 배경색 관련 이슈.

**Why:** 사용자 실기기 테스트에서 발견된 UX 문제들.
**How to apply:** 알람 관련 UI/로직 수정 시 이 규칙들 반드시 확인.
